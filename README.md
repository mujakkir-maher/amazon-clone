# Amazon Clone

A front-end clone of the Amazon homepage, built using plain HTML and CSS. This was a practice project to recreate Amazon's layout — navbar, search bar, category panel, hero banner, product grid, and footer — and make it fully responsive.

Live: https://mujakkir-maher.github.io/amazon-clone/

## What's included

- Navbar with delivery location, search bar, sign-in, returns, and cart sections
- Category panel below the navbar (Today's Deals, Gift Cards, Prime Video, etc.)
- Hero banner section
- Product category grid with 8 sections
- Footer with company links
- Responsive design — works on desktop, tablet, and mobile

## Built with

- HTML5
- CSS3 (Flexbox + media queries for responsiveness)
- Font Awesome for icons

## Running it locally

Clone the repo and just open `index.html` in a browser:

```bash
git clone https://github.com/mujakkir-maher/amazon-clone.git
cd amazon-clone
```

No build steps or dependencies needed since it's plain HTML/CSS.

## Notes

This is a learning project, not affiliated with Amazon in any way. The layout and styling were recreated for practice — assets like images and the logo are placeholders.
